# Dovie
A booking ticket app that allows user to buy ticket with a tap.

## What is included
1. Onboarding SplashScreen
2. Sign In/Up
3. Home Activity
4. Ticket Activity
5. Profile Activity
6. Movie Description
7. Selecting Seats
8. CheckOut
9. Ticket Details
10. Profile Screen

## Utilities
1. Android Studio (Kotlin)
2. Firebase
